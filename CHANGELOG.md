# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## [0.1.0](https://github.com/jmchilton/ghwt/compare/initial...0.1.0) (2025-11-17)

### Features

- Worktree-centered development task dashboard with Obsidian integration
- Git worktree management (create, delete, sync)
- Terminal session management (tmux/zellij)
- GitHub integration with PR detection and fork handling
- Obsidian vault integration with metadata tracking
- Claude Code integration
- CI artifact downloading and analysis
- Interactive worktree picker with fuzzy search
- Agent slash commands for Claude integration
