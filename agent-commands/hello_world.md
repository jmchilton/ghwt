# Hello World

Example slash command for ghwt.

This is a simple test command to verify that the ghwt agent slash command integration is working correctly.

## What This Does

- Demonstrates the basic structure of a ghwt slash command
- Can be used to verify that Claude Code can find and load the command
- Shows how to write helpful documentation for users

## Usage

When you run `/ghwt:hello_world`, Claude will:

1. Load this file
2. Use the content as context for responding to your request
3. Help you with information about ghwt
